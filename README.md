# Pothole.nz

Landing page for **Pothole.nz** — asphalt maintenance and driveway care services in Auckland, New Zealand.

## About

Pothole.nz is a trading name of **One Stop Parking Lot Care Limited** (NZBN: 9429051381493), a small family-run business operating across the Auckland region since 2023. We specialise in pothole repair, crack sealing, and asphalt seal coating for residential driveways and commercial carparks.

## Services

- **Pothole Repair** — professional cold mix application using Promix products. Repair life of 10–15 years.
- **Crack Sealing** — Promix Asphalt Cracksealer for long-term protection against water damage.
- **Asphalt Seal Coating** — Promix SuperSeal application that protects against UV, rain, and tyre abrasion.

## What makes us different

- Trusted Promix Applicator — recommended by ColdMix NZ
- 2-Year Workmanship Guarantee
- 12+ years of combined experience in traffic management, roading, and bitumen products
- Right tool for the job — cold mix specialisation for residential driveways and smaller commercial carparks

## Running locally

This is a static site — no build step required.

```bash
# Open directly in browser
open index.html

# Or serve via a local HTTP server (optional)
python3 -m http.server 8000
# Then visit http://localhost:8000
```
